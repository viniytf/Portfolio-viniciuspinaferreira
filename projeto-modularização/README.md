# 🧩 Modularização de Código
 
## 📝 Descrição do Projeto
Este projeto aborda a aplicação de modularização no desenvolvimento de software, com foco na separação do código em funções independentes para melhorar organização, reutilização e manutenção.

A estrutura modular permite que cada parte do sistema tenha uma responsabilidade específica, tornando o código mais limpo, escalável e fácil de entender.
 
## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python  
* **Conceitos Aplicados:** Funções, escopo, reutilização de código, organização modular e princípio da responsabilidade única  
* **Ferramentas:** VS Code / Terminal
 
## 📊 Funcionalidades
* Separação do código em funções independentes  
* Organização lógica e modular  
* Reutilização de funções  
* Melhor legibilidade e manutenção do código
